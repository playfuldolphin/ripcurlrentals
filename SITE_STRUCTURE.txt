HOW BOOKING WORKS (CURRENT SETUP)
================================================================================

1. Guest fills out booking form with:
   - Name, email, phone
   - Check-in and check-out dates
   - Number of guests
   - Special requests/questions

2. Clicks "Send Inquiry"

3. Opens their email client with pre-filled message containing all info

4. They send the email to you (bookings@riptiderentals.com)

5. You receive the inquiry and respond directly

NOTE: This is a simple email-based system. No payment processing yet.
      You can add Stripe/PayPal integration later.

TYPICAL WORKFLOW
================================================================================

1. Edit src/data/properties.js
   └─ Add your property info and photo URLs

2. Update contact info in 3 files
   └─ Search for "bookings@riptiderentals.com" and replace

3. npm run dev
   └─ Test locally at http://localhost:5173

4. Make adjustments as needed

5. npm run build
   └─ Creates production-ready files in dist/

6. Deploy to Netlify/Vercel
   └─ Your site goes live!

7. Point custom domain (optional)
   └─ yourrentals.com

8. Share link with potential guests
   └─ Start getting bookings!
